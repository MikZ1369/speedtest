# Speed test



![LibreSpeed-Android Logo](./icons/icon.png)

## Compatibility
Android 4.0.3 and up (SDK 15), all architectures.

## Features
* Download
* Upload
* Ping
* Jitter
* IP Address, ISP, distance from server (optional)
* Telemetry (optional)
* Results sharing (optional)
* Multiple Points of Test (optional)
